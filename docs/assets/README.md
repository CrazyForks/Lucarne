# README Demo 素材清单

请补齐下列文件，README 会直接引用：

| 文件 | 类型 | 内容 |
|---|---|---|
| `demo-telegram-panel.gif` | GIF | Telegram 输入 `/panel`，查看 Overview / Workspaces / Sessions，多 Agent 状态可见。 |
| `demo-wechat-quote-context.png` | PNG | 微信引用上一条 Lucarne 通知，AI 接续原 session 上下文。 |
| `demo-telegram-approve-claude.gif` | GIF | Claude 请求执行命令，Telegram 点击 `[Approve]` 后继续执行。 |

建议：

- GIF 宽度 900–1200px，控制在 10MB 内。
- 截图隐藏 token、chat id、用户头像、真实路径等敏感信息。
- Telegram 动图优先展示移动端；微信截图优先展示引用气泡和 AI 回复。
